程序目录及文件说明

根目录
├─bin
│  ├─aix      	Aix版本的KSMBCC主执行程序
│  ├─linux  		Linux版本的KSMBCC主执行程序
│  ├─solaris		Solaris版本的KSMBCC主执行程序
│  ├─bu0    		ksmbcc里面用到的bu0相关配置
│  ├─bu1		ksmbcc里面用到的bu1相关配置
│  └─bu2		ksmbcc里面用到的bu2相关配置
├─install		导入数据库脚本的shell（已废弃）
├─lib			ksmbcc、ykt_bu需要使用的公共lib，发布时需要将子目录对应操作系统版本的lib复制到本目录下。
│  ├─aix		aix版本
│  ├─linux		linux版本
│  └─solaris		solaris版本
├─log			系统运行时的日志目录，会通过sall自动创建
├─sbin		需要发布到系统定时任务中的shell程序
├─sql_initial		数据库初始化sql语句（含建用户、建表等）
│  └─createdb	利用Oracle的建库脚本来创建一卡通数据库（已废弃）
│      └─dbtmpl	建库模板（已废弃）
├─sql_upgrade		一卡通升级SQL脚本。请根据当前ykt_bu版本来决定需要执行的SQL
├─src			源代码目录
│  ├─batch		独立运行程序。包括入账程序、银行对账程序、电子钱包解锁程序等。
│  ├─buinc		公共bu使用的.h
│  ├─bupub		公共bu使用的.cpp
│  ├─db		数据库表的映射配置文件，会自动生产相关代码
│  ├─func		全部业务功能号
│  ├─include		功能号使用的.h
│  │  └─dbincl	根据act自动生成的.h
│  ├─public		公共的.cpp
│  ├─test		python的测试脚本
│  ├─tool		dbcrt的源代码
│  ├─watch		老版本的监控服务端程序，应该没有客户使用了。
│  └─xpack		xpack源代码
├─test		导入导出数据库的shell（停止维护）
├─yktv3release	codeblocks的工程文件目录
├─AutoInstall.sh	linux系统下自动创建ykt用户并调用Install_v3.sh来安装一卡通服务端程序的shell
├─compile&deploy.txt	编译和发布说明
├─compile.sh		自动编译shell
├─filelist.txt	本文件
├─Install_v3.sh	安装一卡通服务端程序的shell
├─ReleaseLog.txt	修改日志
├─ykt_deploy.sh	一卡通制作安装包和升级包的shell
├─yktprofile		一卡通用户需要使用的profile
├─yktsetup.sh		老版本的基于源代码的安装脚本（已废弃）
└─setup.sh		老版本的基于源代码的安装脚本（已废弃）
