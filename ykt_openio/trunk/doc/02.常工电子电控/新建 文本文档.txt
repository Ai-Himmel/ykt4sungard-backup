    接口函数：
            function CNectDB(bIsConnect : boolean): Single; stdcall;
    bIsConnect ＝true；表示数据库连接；false，表示断开连接。
    返回值 系统电量单价，单位为分。为0表示执行失败。

            function GetRoomName(nRoomCode : integer; sRoomName : pchar): integer; stdcall;
    nRoomCode 为房间编码：为六位数。最高位为楼幢属性，例如 1表示男生，2表示女生等
            次两位为楼幢编码，例如 A2 楼编号为32等 后三位为房间编号 如306室，即306
            具体编码由学校确定！
    sRoomName 为返回的房间全名 格式如 男生A2 楼306室
            其长度不超过80个字符，空间由函数调用方预先分配。
     返回值 1 成功，其他暂不定义。

            function SellElec(nRoomCode, nMoney : integer; sName : pChar): integer; stdcall;
    nRoomCode 同上
    nMoney       购电金额，单位为分
    sName        购电人姓名 长度不超过10字符（5汉字）
    返回值         同上
            function GetDBStatus(): integer; stdcall;
    获取dll与数据库连接状态：无输入参数！
    返回值 1 表示正常连接，其他暂不定义。
