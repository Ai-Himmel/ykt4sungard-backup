【组件包内，包括如下文件】
-------------------------


jsdykt.dll			购电接口动态链接库升级（供第三方接口调用文件）


【附件】

jsd_test.exe			购电接口动态库测试程序

购电接口升级安装说明.txt	安装说明(本说明文件)




【升级安装过程】
-----------

1、请确认原购电接口在SQLSERVER版本的SIMS电控系统下能够正常运行！如果原接口没有正常运行，请先安装原购电接口，再升级！

2、将jsdykt.dll文件拷贝到原jsdykt.dll所在的安装文件夹下，覆盖原文件；

3、打开购电接口动态链接库的配置文件jsdykt.ini，增加如下项：

   [SIMS]
   Server=	SIMS数据库所在服务器名
   Name=	SIMS数据库登录用户名
   Pswd=	SIMS数据库用户密码


4、可以通过运行购电接口动态库测试程序jsd_test.exe，测试通过jsdykt.dll进行的读取房间剩余电量的操作！

5、jsdykt.dll新增的读取房间剩余电量接口函数定义如下：

   function GetRemainE(nRoomCode : integer; sElecDate : pchar): integer; stdcall; external 'jsdykt.dll';

   // nRoomCode  房间编码，整型，输入项
   // sElecDate  电量采集时间，字符串，输出项
   // 函数返回值 剩余电量，整数，单位0.01度，输出项
   //     定义： -99999  表示该房间没有采集到数据
   //            -100000 表示没有读取到该房间电量数据，或者函数执行出错




【可能的报错及处理】
-------------------

1、此次升级的购电接口，在运行时将自动连接SIMS电控系统的数据库！如果连接不上，则无法进一步操作！如果有问题，请及时与我公司开发部联系！
