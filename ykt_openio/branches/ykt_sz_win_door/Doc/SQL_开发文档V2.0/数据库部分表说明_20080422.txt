发布时间: 2008-4-22

t_d_CardRecord 刷卡记录  (放在ACCESS和SQL Server的表中)
  各字段简要说明
	f_CardRecordID  记录序列号
	f_CardNO  			刷卡的卡号
	f_ReadDate  		刷卡的日期时间
	f_Character 		刷卡的有效或无效(1表示有效, 0表示无效)
	f_ControllerSN  控制器号
	f_ReaderID 			刷卡所在的读卡器序列号
	f_ConsumerID 		用户序列号
	f_WarnType  		报警类型
	f_Note      		备注

v_d_CardRecord 刷卡详细记录视图 (放在ACCESS的查询, SQL Server的视图中)
  主要字段说明
	f_CardRecordID  记录序列号
	f_CardNO  			刷卡的卡号
	f_ReadDate  		刷卡的日期时间
	f_Character 		刷卡的有效或无效(1表示有效, 0表示无效)
	f_ControllerSN  控制器号
	f_ReaderID 			刷卡所在的读卡器序列号
	f_ConsumerID 		用户序列号
  f_ConsumerName  用户姓名
  f_GroupName     部门班组
  f_ConsumerNO    用户编号
  f_ControllerID  控制器在数据库中的序列号
  f_ReaderName    刷卡所在的读卡器名称
  
v_b_ConsumerIDCard 用户-部门-卡号视图 (放在ACCESS的查询, SQL Server的视图中)
  主要字段说明
  f_ConsumerID    用户序列号
  f_ConsumerNO    用户编号
  f_ConsumerName  用户姓名
	f_CardNO  			用户卡号
  f_WorkNo        用户工号
  f_GoupID        部门班组ID
  f_GroupName     部门班组名称
  f_AttendEnabled 用于考勤
  f_DoorEnabled   用于门禁
