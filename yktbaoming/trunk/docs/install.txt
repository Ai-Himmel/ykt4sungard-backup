1.配置jndi，增加数据库连接，连接名为 jndi/baoming
tomcat的配置方法：
在$TOMCAT_HOME/conf/Catalina/localhost下添加5baoming.xml，文件内容：

<Context path="/baoming" docBase="/opt/sungard/webapps/baoming" debug="99" reloadable="true">
    <Logger className="org.apache.catalina.logger.FileLogger"
        prefix="log_baoming." suffix=".txt" timestamp="true"/>

    <Resource name="jdbc/baoming" auth="Container" type="javax.sql.DataSource"/>
    <ResourceParams name="jdbc/baoming">
                        <parameter>
                     <name>factory</name>
                     <value>org.apache.commons.dbcp.BasicDataSourceFactory</value>
                   </parameter>
        <!-- Maximum number of dB connections in pool. Make sure you
             configure your mysqld max_connections large enough to handle
             all of your db connections. Set to 0 for no limit.
             -->
        <parameter>
            <name>maxActive</name>
            <value>100</value>
        </parameter>
        <!-- Maximum number of idle dB connections to retain in pool.
             Set to 0 for no limit.
             -->
 <parameter>
            <name>maxIdle</name>
            <value>30</value>
        </parameter>
        <!-- Maximum time to wait for a dB connection to become available
             in ms, in this example 10 seconds. An Exception is thrown if
             this timeout is exceeded.  Set to -1 to wait indefinitely.
             -->
        <parameter>
            <name>maxWait</name>
            <value>10000</value>
        </parameter>
        <!-- DB2 dB username and password for dB connections  -->
        <parameter>
            <name>username</name>
            <value>ykt_cur</value>
        </parameter>
        <parameter>
            <name>password</name>
            <value>kingstar</value>
        </parameter>
 <!-- Class name for JDBC driver -->
        <parameter>
            <name>driverClassName</name>
            <value>oracle.jdbc.driver.OracleDriver</value>
        </parameter>

        <parameter>
            <name>dialect</name>
            <value>org.hibernate.dialect.Oracle9Dialect</value>
        </parameter>

        <parameter>
            <name>url</name>
            <value>jdbc:oracle:thin:@192.168.56.92:1521:yktdb
                          </value>
        </parameter>
        <Resource name="UserTransaction" auth="Container" type="javax.transaction.UserTransaction"/>
 <parameter>
              <name>removeAbandoned</name>
              <value>true</value>
         </parameter>

        <parameter>
            <name>removeAbandonedTimeout</name>
            <value>60</value>
        </parameter>
        <!-- Log a stack trace of the code which abandoned the dB
             connection resources.
             -->
        <parameter>
            <name>logAbandoned</name>
            <value>true</value>
        </parameter>
    </ResourceParams>
</Context>

数据库URL、用户名、密码请自行修改。

2、修改WEB-INF/conf/config.properties中的数据库相关配置信息。

hibernate.dialect org.hibernate.dialect.OracleDialect
hibernate.connection.driver_class oracle.jdbc.driver.OracleDriver
hibernate.connection.username ykt_cur
hibernate.connection.password kingstar
hibernate.connection.url jdbc:oracle:thin:@192.168.56.92:1521:yktdb
