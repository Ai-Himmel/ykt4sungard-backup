1、请修改 client.properties 配置信息
2、在Tomcat下部署这个Web应用就可以应用了
3、index.jsp和search.jsp实现了一个简单的认证头，在search中根据请求返回显示属性值。在这个简单的程序中用到了多个认证函数
	GetLoginURL() 得到登录IS认证的URL
	GetLogoutURL() 得到从IS注销的URL
	GetCurrentUser() 根据SSOToken值得到当前的用户
	GetUserAttribute() 根据参数得到用户的属性
