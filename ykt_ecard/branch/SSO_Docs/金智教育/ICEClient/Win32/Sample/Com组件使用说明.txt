使用说明：
1、首先要部署COM组件认证接口，正确配置Client.properties，然后注册
2、然后拷贝index.aspx和search.aspx到虚拟目录下的任意一个文件夹下
3、启动IIS，访问index.aspx即可应用
4、index.aspx和search.aspx实现了一个简单的认证头，在search中根据请求返回显示属性值。在这个简单的程序中用到了多个认证函数
	GetLoginURL() 得到登录IS认证的URL
	GetLogoutURL() 得到从IS注销的URL
	GetCurrentUser() 根据SSOToken值得到当前的用户
	GetUserAttribute() 根据参数得到用户的属性
5、test.asp中对认证接口中的大部分函数给出了一个简单的示例，在应用中可以参考适用，对于程序中的接口函数参考《Com API 使用说明》
